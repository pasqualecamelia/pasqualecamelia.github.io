# Data provenance — CamH

## Generated data
All CSV/JSON files are numerical outputs of `run_camH_validation.py`,
recomputed from the monograph's structural constants (φ, Γ_τ, δ_F, the
B₁₃=S₅⊕T₈ ribbon split). No third-party data is bundled.

## Reference values
CODATA 2018 constants (α⁻¹, R∞, a₀, mₑ) are used only for the hydrogen
metrology cross-check (appB Bohr–Rydberg gate). They are public reference
values, not redistributed datasets.
