# Data provenance

## Included third-party data

Planck 2018 PR3 CMB power spectra (Planck Legacy Archive, Release R3.01):
  COM_PowerSpect_CMB-TT-full_R3_01.txt
  COM_PowerSpect_CMB-EE-full_R3_01.txt
  COM_PowerSpect_CMB-TE-full_R3_01.txt

Column format: ell  D_ell [muK^2]  sigma_minus  sigma_plus

These files are unchanged from the Planck Legacy Archive distribution.
MD5 checksums are listed in MD5SUMS.txt.

## Generated data

The following files are generated by `run_qgt_gate3_camb.py` using
CAMB v1.6.6 with the QGT structural parameter vector:

  camb_qgt_gate3.csv   — QGT Gate-3 TT/EE/TE spectra
  camb_lcdm_ref.csv    — LCDM reference TT/EE/TE (Planck 2018 best-fit)
  camb_lcdm21_ref.csv  — LCDM reference TT (Planck 2021 PR4 best-fit)
  metrics_gate3.json   — chi2_nu, RMS, structural parameter values
  fig_cmb_gate3_final.png — Gate-3 figure (TT/EE/TE vs Planck 2018)

The PR4 reference files (COM_PowerSpect_CMB-*-full_R4_PR4.txt) are
CAMB-generated curves from the Tristram+2021 best-fit parameter vector.
They are not official PR4 binned spectra.

These generated files are numerical outputs of the CamCMB v176 diagnostic
pipeline. They are not official Planck products.
